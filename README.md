## 🧐 About
This project started because I thought I could do in 2 hours what a well-documented library already does in 5 minutes. It’s built on caffeine, Stack Overflow snippets, and a refusal to read the official documentation until absolutely necessary.

**What it actually does:**
* Solves a problem you probably didn't have.
* Consumes more RAM than Chrome with 50 tabs open.
* Makes me look busy during Zoom calls.

## 🛠 Prerequisites
Before you try to run this, ensure you have:
* **Patience:** At least 2.0GB.
* **Coffee:** Strong brew, preferably dark roast.
* **Python/Node/Go:** Whatever version I used (I can't remember, try them all).
